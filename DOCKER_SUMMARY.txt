================================================================================
                        TROUBLESHOOTING
================================================================================

If services don't start:
  1. Check Docker is running
  2. Run: docker-compose logs
  3. Rebuild: docker-compose down && docker-compose up -d --build

If database won't connect:
  1. Wait 10-15 seconds for MySQL to initialize
  2. Check: docker-compose logs db
  3. Verify health: docker-compose exec db mysqladmin ping -u root -ppassw0rd

If frontend can't reach backend:
  1. Check backend is running: docker-compose ps
  2. Test: curl http://localhost:5000/hello
  3. Check logs: docker-compose logs backend

Port already in use:
  1. Edit docker-compose.yml and change port mapping
  2. Or stop other services using those ports
